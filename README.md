 Project Overview
I created a 5-page website for a clothing brand called "Brotherhood".  
The website is meant to give the brand an online presence and allow users to view products and contact the business.

 Website Goals
- Create a clean and simple website  
- Improve brand visibility  
- Make it easy for users to navigate  
- Show products clearly  
- Allow users to send enquiries  

 What I Included
- Navigation menu on all pages  
- Homepage (index.html)  
- About page  
- Products page  
- Enquiry form  
- Contact page
